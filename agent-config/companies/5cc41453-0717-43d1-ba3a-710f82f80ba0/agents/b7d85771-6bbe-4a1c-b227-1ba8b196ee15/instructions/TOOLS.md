# Tools

## Notion MCP (`claude.ai Notion`)

Connected via `claude.ai Notion` MCP server. Available on every run.

Key tools for the plan review workflow:

| Tool | When to use |
|---|---|
| `mcp__claude_ai_Notion__notion-get-comments` | Fetch inline board comments from a Notion plan page |
| `mcp__claude_ai_Notion__notion-create-pages` | Write a new plan to Notion for board review |
| `mcp__claude_ai_Notion__notion-fetch` | Read an existing Notion page |
| `mcp__claude_ai_Notion__notion-update-page` | Update a Notion page after incorporating feedback |
| `mcp__claude_ai_Notion__notion-search` | Find Notion pages by name |
| `mcp__claude_ai_Notion__notion-create-comment` | Add a comment to a Notion page |

See `NOTION-REVIEW-WORKFLOW.md` in the project root for the full plan review protocol.
