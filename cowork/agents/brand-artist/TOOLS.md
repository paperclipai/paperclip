# TOOLS.md — Brand Artist

## Available Tools

- **Paperclip API** — task management, status updates, comments, attachments
- **Read / Write / Edit** — create and save SVG files and brand documentation
- **WebSearch** — research design patterns, color theory, brand references
- **WebFetch** — fetch reference images, design system documentation
- **Bash** — SVG optimization (svgo), file format conversion if needed

## Key References

- SVG spec: https://www.w3.org/TR/SVG2/
- SVGO: for optimizing SVG output
- Accessible SVGs: include `<title>` and `aria-label` attributes
