# @paperclipai/adapter-qwen-local

## 0.1.0

### Minor Changes

- Initial release: Qwen Code (local) adapter for Paperclip
- Supports `qwen --output-format stream-json` with session resume, `--yolo` mode, and model selection
- Models: qwen3-coder-plus, qwen3-coder, qwen3-coder-flash, qwen-plus, qwen-turbo
- Environment test with hello probe and DASHSCOPE_API_KEY detection
