# Requirements: RealTycoon2

**Defined:** 2026-05-04
**Core Value:** 회사 범위 work signal은 disconnected tool이나 Paperclip-shaped manual workflow를 강요하지 않고 logging -> execution -> knowledge accumulation -> approval -> economic feedback으로 이어져야 한다.

## v1 Requirements

Requirements for initial release. Each maps to roadmap phases.

### Trello Field Extension Parity

- [ ] **TRELLO-01**: User can add checklist items to a card and track completion percentage
- [ ] **TRELLO-02**: User can set and edit due date with time on a card
- [ ] **TRELLO-03**: User can attach colored labels to cards for visual categorization
- [ ] **TRELLO-04**: User can assign/remove members to/from a card

### Trello Automation / Power-up Parity (1)

- [ ] **POWER-01**: User can create custom fields on a board (text, number, date, dropdown types)
- [ ] **POWER-02**: User can view and edit custom field values directly on card
- [ ] **POWER-03**: User can use a board calendar view filtered by due date

### Trello Automation / Power-up Parity (2)

- [ ] **POWER-04**: User can create formula-based fields on cards (arithmetic, date differences)
- [ ] **POWER-05**: User can set board-level card limits per lane (WIP limits)
- [ ] **POWER-06**: User can use card templates for quick card creation
