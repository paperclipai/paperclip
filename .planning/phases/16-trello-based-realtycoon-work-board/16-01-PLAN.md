# Phase 16 Plan: Trello-Based RealTycoon Work Board

## 목표

기존 issue board처럼 보이던 메인 업무 표면을 RealTycoon2의 Trello형 업무 보드로 전환한다. 내부 compatibility layer는 유지하되, 사용자는 Task/To-Do 카드, lane 이동, 빠른 편집, 산출물/가격/OKR badge를 중심으로 일한다.

## 작업

- [x] `/issues` 기본 진입을 RealTycoon2 업무 보드로 바꾸고 storage key를 legacy Paperclip key와 분리한다.
- [x] `KanbanBoard` 카드를 RealTycoon2 Task/To-Do 카드로 고도화한다.
- [x] 카드에 산출물 count, 가격 badge, OKR badge, To-Do count, 담당자 정보를 표시한다.
- [x] drag/drop status 이동을 유지하면서 카드 내 lane/status 및 priority quick edit를 추가한다.
- [x] lane/header에서 새 작업 생성 시 status default를 전달한다.
- [x] One-Liner inbound draft source contract를 messenger/mobile/native까지 확장하고 검수 가능한 route를 화면에 표시한다.
- [x] 관련 컴포넌트/contract 테스트를 추가/갱신한다.

## 완료 기준

- `TRELLO-01`: Task/To-Do 카드, lane, drag/drop, 빠른 편집, 산출물/가격/OKR badge가 메인 workflow에 존재한다.
- `TRELLO-02`: 사용자가 보는 `/issues` 메인 표면은 RealTycoon2 업무 보드이며 legacy issue board 느낌이 줄어든다.
- `CAPTURE-01`: One-Liner inbound draft contract가 `slack`, `teams`, `webhook`, `mobile`, `native` source를 수용하고, One-Liner 화면이 검수 가능한 route를 표시한다.
