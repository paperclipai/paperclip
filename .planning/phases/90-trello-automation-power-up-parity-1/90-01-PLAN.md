### Task 90-01-1: Create custom field schema tables

**Type:** `schema`
**Verify:** `pnpm db:generate` produces migration for new tables

- [ ] Read `packages/db/src/schema/rt2_work_board.ts`
- [ ] Add `rt2WorkBoardCustomFields` table:
  ```typescript
  rt2WorkBoardCustomFields = pgTable("rt2_work_board_custom_fields", {
    id: uuid("id").primaryKey().defaultRandom(),
    companyId: uuid("company_id").notNull().references(() => companies.id, { onDelete: "cascade" }),
    name: text("name").notNull(),
    fieldType: text("field_type").notNull().default("text"), // text | number | date | dropdown
    position: integer("position").notNull().default(0),
    createdByUserId: text("created_by_user_id"),
    createdAt: timestamp("created_at", { withTimezone: true }).notNull().defaultNow(),
    updatedAt: timestamp("updated_at", { withTimezone: true }).notNull().defaultNow(),
  }, (table) => ({
    companyPositionIdx: index("rt2_work_board_custom_fields_company_position_idx").on(table.companyId, table.position),
  }));
  ```
- [ ] Add `rt2WorkBoardCustomFieldOptions` table:
  ```typescript
  rt2WorkBoardCustomFieldOptions = pgTable("rt2_work_board_custom_field_options", {
    id: uuid("id").primaryKey().defaultRandom(),
    companyId: uuid("company_id").notNull().references(() => companies.id, { onDelete: "cascade" }),
    fieldId: uuid("field_id").notNull().references(() => rt2WorkBoardCustomFields.id, { onDelete: "cascade" }),
    label: text("label").notNull(),
    position: integer("position").notNull().default(0),
    createdAt: timestamp("created_at", { withTimezone: true }).notNull().defaultNow(),
  }, (table) => ({
    fieldPositionIdx: index("rt2_work_board_custom_field_options_field_position_idx").on(table.companyId, table.fieldId, table.position),
  }));
  ```
- [ ] Add `rt2WorkBoardCardCustomFieldValues` table:
  ```typescript
  rt2WorkBoardCardCustomFieldValues = pgTable("rt2_work_board_card_custom_field_values", {
    id: uuid("id").primaryKey().defaultRandom(),
    companyId: uuid("company_id").notNull().references(() => companies.id, { onDelete: "cascade" }),
    issueId: uuid("issue_id").notNull().references(() => issues.id, { onDelete: "cascade" }),
  ```
