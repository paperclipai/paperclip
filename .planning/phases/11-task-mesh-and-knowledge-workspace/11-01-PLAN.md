# Phase 11 Plan: Task Mesh and Knowledge Workspace

**작성일:** 2026-04-25
**상태:** 실행 완료

## 목표

개발기획서의 Task Mesh와 wiki/graph/Obsidian-ready knowledge workflow를 기존 RealTycoon2 spine 위에 연결한다.

## 작업

1. Shared contract 확장
   - graph node/edge type에 deliverable/actor/event 및 projector edge를 포함한다.
   - Task Mesh 7개 view와 node evidence contract를 추가한다.
   - vault export contract를 추가한다.

2. Server read model 확장
   - `rt2-task-mesh`가 task별 deliverable, owner, execution, quality, gold, knowledge evidence를 계산한다.
   - graph report가 God Node, surprising connection, stale warning을 반환한다.
   - knowledge projector가 pending event와 last projection time을 반환한다.
   - Obsidian-compatible vault export route를 추가한다.

3. UI 연결
   - `Rt2GraphPanel`을 7개 Task Mesh view로 전환 가능하게 만든다.
   - Knowledge page의 graph tab을 실제 Task Mesh panel에 연결한다.
   - Knowledge page에서 wiki page list와 vault export preview를 보여준다.

4. 검증
   - shared/server/ui typecheck.
   - knowledge projector/routes Vitest.

## 성공 기준
