# Phase 14: Daily Kanban Trello Parity - Plan

**상태:** Complete
**Requirement IDs:** DAILY-BOARD-01, DAILY-BOARD-02

## 목표

일일업무일지의 3칸 보드를 Trello처럼 직접 조작 가능한 업무 보드로 만든다. 사용자는 To-Do 카드를 drag/drop으로 다른 칸에 옮길 수 있고, 이동 결과는 기존 daily report card save API를 통해 즉시 저장되어야 한다.

## 범위

- `Rt2DailyBoard`에 HTML drag/drop 기반 카드 이동을 추가한다.
- 카드 이동 시 `onSaveCard`를 즉시 호출해 기존 저장 경로를 재사용한다.
- 기존 lane select와 수동 저장 버튼은 유지한다.
- 컴포넌트 테스트로 3칸 렌더링, lane select 저장, drag/drop 저장을 검증한다.

## 완료 기준

- [x] `DAILY-BOARD-01`: 사용자는 3칸 칸반에서 To-Do 카드를 drag/drop으로 다른 칸에 옮길 수 있다.
- [x] `DAILY-BOARD-02`: 카드 drop 시 daily report save contract로 lane 변경이 즉시 저장된다.
