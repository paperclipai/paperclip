### Task 91-01-6: Create card template schema

**Type:** `schema`
**Verify:** Migration generated

- [ ] Create `rt2WorkBoardCardTemplates` table:
  ```typescript
  rt2WorkBoardCardTemplates = pgTable("rt2_work_board_card_templates", {
  ```
