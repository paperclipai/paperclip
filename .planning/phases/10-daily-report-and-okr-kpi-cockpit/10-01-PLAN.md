# Phase 10 Plan 01: Daily Report and OKR/KPI Cockpit

**상태:** 완료
**날짜:** 2026-04-25

## 목표

Project `Daily` tab을 RealTycoon2 일일보고 cockpit으로 확장하고, daily report API가 작업/산출물/OKR/KPI/gold/XP/gap 정보를 함께 제공하게 만든다.

## 작업

1. `Rt2DailyBoard` shared contract 확장
   - card별 deliverable count, submitted count, base price total, quality status, OKR context status, gap flags 추가.
   - board-level `cockpit.summary`, `traceRows`, `gapFlags`, `aiSummary` 추가.

2. daily report service read model 확장
   - assigned to-do와 parent task를 기준으로 `issueWorkProducts` 산출물 metadata 집계.
   - task/project goal context를 읽어 goal parent path 구성.
   - progress/to-do status와 산출물 base price에서 bottom-up KPI/gold/XP summary 산출.

3. UI cockpit 전환
   - 왼쪽: report date, summary metric, OKR/KPI trace.
   - 가운데: 기존 three-lane report/task editor.
   - 오른쪽: Jarvis summary, gold/XP/quality, 보완 gap.

4. 검증
   - shared/server/ui typecheck.
   - `Rt2DailyBoard` UI test.
   - `rt2-daily-report-routes` integration test.
