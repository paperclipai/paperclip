# Top Instagram Page Content System

## Goal
The AI organisation must copy the workflow used by top-level Instagram pages, but perform it using AI agents and automation.

## Top Pages Manual Workflow
Top pages do not create random content. They follow a factory:

1. Research audience pain
2. Study competitors
3. Find trends and repeating questions
4. Choose a strong emotional angle
5. Write a powerful hook
6. Write script or carousel copy
7. Design visual scenes
8. Add voice/audio/music direction
9. Edit into final media
10. Add caption and hashtags
11. QA for brand/safety/originality
12. Upload
13. Track analytics
14. Improve next content

## AI Organisation Version
The same workflow becomes:

Research Agent -> Audience Psychology Agent -> Competitor Study Agent -> Strategy Agent -> Hook and Script Agent -> Media Direction Agent -> Audio/Voice Agent -> Media Worker -> Caption Agent -> QA Agent -> Growth Agent -> Memory Agent.

## Success Metric
The organisation is not successful when it writes good text.

The organisation is successful when it produces a ready-to-upload media pack:

- final_reel.mp4
- cover image
- voiceover audio
- subtitles/text
- caption
- hashtags
- QA report
- learning note

## Content Formula
Every reel should include:

1. Hook
2. Indian relatable situation
3. Emotional/social tension
4. Mature insight
5. Payoff / shareable line
6. CTA or save/share reason

## SINK/DINK Reel Formula
Hook -> family/social pressure -> couple perspective -> respectful insight -> personal timeline message -> shareable ending.

## Avoid Generic AI Content
The organisation must avoid:

- generic motivational lines
- bland AI-sounding captions
- copied trend text
- unoriginal repost-style content
- parenting/kids drift

## Required Output Quality
Every content idea should be transformed into a media-first plan:

- Scene-by-scene structure
- Overlay text
- Voiceover
- Visual direction
- Audio mood
- Caption
- Hashtags
- QA note

## Learning Loop
After every output:

- Save what worked
- Save what failed
- Save rejected content reason
- Save best hooks
- Save brand drift examples
- Use this in future prompts
