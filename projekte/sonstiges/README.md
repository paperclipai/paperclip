# Sonstiges

**Worum geht's:** Sammelordner für alles, was keinem der sechs Haupt-Subprojekte zugeordnet ist.

**Unterordner:**
- `cloudflare/` — Cloudflare-Umzug, Whitepaper, Access-Konfiguration
- `paperclip-allgemein/` — übergreifende Paperclip-Themen (Anleitung, Anpassungen, LLM-Empfehlungen, Agent-Setup, Chiefs-Delegation)
- `archiv/` — ältere Artefakte, SQL-Dumps, nicht mehr aktive Dateien

Wenn ein Thema hier wächst und eigenständig wird, kann es zu einem eigenen Subprojekt-Ordner promoviert werden.
