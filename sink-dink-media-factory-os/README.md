# SINK DINK AI Media Factory OS

This folder defines the Paperclip-native media factory organisation for SINK DINK.

## Mission
Build Paperclip into an AI Media Factory that can produce upload-ready media packs without requiring a separate workflow tool.

The user should only need to give a production command, review the output, and manually upload final files.

## Core Direction
Paperclip is the command center. GitHub stores durable system specs and templates. Render runs the server. Agents execute work through tasks, skills, routines, tools, plugins, and artifacts.

## Existing Paperclip Foundation
Paperclip already has agents, tasks, subtasks, skills, routines, artifacts, approvals, plugins, adapters, projects, goals, costs, activity logs, secrets/env, and an internal API.

## What This OS Adds
- Internal Automation Orchestrator
- Media File Engine
- Reel video generation pipeline
- Carousel image generation pipeline
- TTS and subtitle pipeline
- Connector framework
- AI model routing policy
- Self-learning daily improvement loop
- QA and approval system
- Upload-ready output pack standard
