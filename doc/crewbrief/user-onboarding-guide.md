# CrewBrief — User Onboarding Guide

**Version:** 1.0 (pre-launch framework)
**Owner:** Sydney, Director of Customer Success
**Status:** Template — populate with product-specific content when app is built

## Overview

CrewBrief delivers premium aviation briefings for pilots and cabin crew. This guide takes a new user from signup to their first successful briefing in under 5 minutes.

---

## 1. Welcome & Account Setup

### Signup Flow
1. User lands on CrewBrief signup page
2. Enters email, creates password, confirms role (pilot / cabin crew / dispatcher)
3. Verifies email via confirmation link
4. Completes profile: airline, aircraft type, home base, languages
5. Accepts Terms of Service and Privacy Policy

### First-Time Setup Checklist
- [ ] Profile complete (airline, aircraft, base)
- [ ] Preferred briefing format selected (text, audio, PDF)
- [ ] Notification preferences configured (email, SMS, in-app)
- [ ] Schedule set (which flights / days to receive briefings)
- [ ] Team/roster connected (if applicable)

## 3. First Briefing

### Step-by-Step

1. **Select flight** — choose from upcoming schedule or enter manually
2. **Configure briefing** — set depth (standard / detailed / executive), include METAR/TAF, NOTAMs, fuel calculations, crew notices
3. **Confirm delivery** — choose when and how to receive the briefing
4. **Receive and review** — briefing appears in the Briefings section
5. **Provide feedback** — rate accuracy and completeness

### Example: Creating Your First Briefing

```
1. Open the Dashboard
2. Click "Schedule Briefing"
3. Select flight "AA1234 — JFK→LAX"
4. Choose "Standard" depth
5. Set delivery: "2 hours before departure" via "Push notification"
6. Click "Schedule"
7. Briefing arrives automatically at the scheduled time
```

## 4. Key Features

### Briefing Formats
- **Standard** — essential weather, NOTAMs, route info (2-3 min read)
- **Detailed** — full METAR/TAF, winds aloft, alternate analysis (5-7 min)
- **Executive** — executive summary with risk assessment (1 page)

### Delivery Channels
- In-app push notification
- Email (HTML or plain text)
- SMS (summary only)
- PDF download

### Crew Collaboration
- Share briefings with crew members
- Add crew-specific notes
- Briefing acknowledgement tracking

## 6. Common Questions

**Q: How far in advance should I schedule a briefing?**
A: Minimum 3 hours before departure for standard briefings, 6 hours for detailed.

**Q: Can I get briefings for non-scheduled flights?**
A: Yes — use "Manual Entry" in the Schedule section.

**Q: What if I need to change my briefing after scheduling?**
A: Edit or cancel from the Schedule section up to 1 hour before delivery.

**Q: Is CrewBrief available offline?**
A: Briefings are cached on receipt; view previously delivered briefings offline.

## 7. Troubleshooting

| Problem | Solution |
|---|---|
| Briefing not delivered | Check notification settings, ensure app has background refresh enabled |
| Missing weather data | Verify aircraft type and route are entered correctly |
| App crashes on briefing view | Force close and reopen; contact support if persistent |
| Can't find my airline | Use "Request Airline" in profile settings |
