The cockpit replaces a dozen growth tools with one surface. Everything an agent touches, you can see and steer from the same place.

## Surfaces

- **Pipeline** — HubSpot deals grouped by stage, forecast, renewals, win/loss, attribution.
- **Outbound** — campaigns ranked by reply rate, multi-sender across email + LinkedIn, AI reply drafts.
- **Inbound** — mentions, demos, deals captured and scored; share-of-voice across LLMs.
- **Content Studio** — idea inbox, briefs, blog + LinkedIn drafts, GSC rank, content gaps.
- **Revenue** — forecast, invoicing, renewals, win/loss analysis.
- **Work OS** — issues, routines, goals, approvals — humans and agents in one orchestration layer.

## One login, every channel

The cockpit is backed by the `/api/agnb` API. Agents and the board operator read and write the same data, so what an agent does is immediately visible to you — and what you approve immediately unblocks the agent.
