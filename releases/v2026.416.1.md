# v2026.416.1

> Released: 2026-04-16

## Security

- **Cross-workspace IDOR in workspace runtime** — `stopRuntimeServicesForExecutionWorkspace` now scopes queries by workspace ownership, preventing agents from stopping services in workspaces they do not own.
- **RCE bypass via `workspaceRuntime` injection** — Agent `PATCH /api/agents/:id` and issue/project `PATCH` routes no longer accept `adapterConfig.workspaceRuntime` writes, closing a path where a malicious agent could inject arbitrary commands into the workspace runtime job spec.
- **Path traversal in `resolveRepoManagedWorkspaceCommand`** — Added null-byte and `../` traversal guards; symlink resolution is bounded to the workspace root, preventing escape to the host filesystem.
- **`buildWorkspaceCommandEnv` sanitization** — Environment variables passed to workspace commands are now sanitized; `PAPERCLIP_WORKTREES_DIR` is forwarded correctly without exposing unrelated host env.

## Fixes

- **Reconciliation test reliability** — Fixed `executionWorkspace` and `projectId` row insertion order in reconciliation tests to prevent foreign-key failures under concurrent test runs.

## Upgrade Guide

No database migrations in this release. This is a security patch — all self-hosted deployments on v2026.416.0 should upgrade.
