# v2026.414.0

> Released: 2026-04-14

## Highlights

- **Self-hosted deploy infrastructure (F6)** — Full production Docker Compose stack with Nginx reverse proxy, PostgreSQL, Redis, and app container. Includes interactive `setup.sh` installer for guided deployment.
- **Docusaurus documentation site** — New `website/` directory with full Portuguese-language documentation covering installation, API reference, FAQ, health monitoring, and backup/restore procedures.
- **Toca da IA branding** — Updated favicon, site manifest, UI index, and branding assets to reflect the Toca da IA identity.

## Added

- `docker/docker-compose.prod.yml` — Production Docker Compose stack with health checks and required env var validation.
- `docker/nginx/conf.d/app.conf` — Nginx HTTP reverse proxy config with WebSocket support and upstream health check passthrough.
- `docker/nginx/conf.d/app-https.conf.example` — HTTPS server block example for TLS deployments.
- `docker/nginx/nginx.conf` — Nginx base config with `$connection_upgrade` map for WebSocket handling.
- `setup.sh` — Interactive bash installer that configures `.env`, validates dependencies, and starts the stack.
- `scripts/backup-docker.sh` — Docker-aware database backup script with tiered retention.
- `.env.prod.example` — Production environment variable template with all required keys documented.
- `website/` — Docusaurus v3 documentation site with docs for install, API, FAQ, deploy operations.
- `docs/deploy/` — Markdown deploy guides (installation, health monitoring, backup/restore) for in-repo reference.

## Improvements

- **Deploy reliability** — Health check URL corrected, Docusaurus static asset paths fixed, nginx `$connection_upgrade` map added for proper WebSocket termination.
- **HTTPS support** — Added port 443 server block example with TLS termination at Nginx.
- **Branding** — CLI feedback report, UI pages, and adapter config fields updated for Toca da IA identity.

## Fixes

- `cli/src/commands/client/feedback.ts` — Updated branding assertion in feedback report.
- `server/src/ui-branding.ts` — Updated favicon SVG to Toca da IA house+dot motif.
- Various UI components — Consistent branding references across `App.tsx`, adapters, and page components.

## Upgrade Guide

This release adds optional self-hosted deploy infrastructure. No database migrations are included. Existing deployments are unaffected — the new Docker Compose and Nginx files only activate when explicitly used.

To deploy with the new stack:
```bash
./setup.sh
```
See `docs/deploy/guia-instalacao.md` for full instructions.
