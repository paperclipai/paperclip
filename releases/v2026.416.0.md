- **Font CDN independence** — UI fonts are now bundled locally instead of loaded from Google Fonts, improving privacy and offline reliability.
- **Backup reliability fix** — `backup-docker.sh` now correctly captures `PIPESTATUS` before it is reset by subsequent commands.
- **Upstream sync** — Merged upstream `paperclipai/paperclip` master (commits through `7f893ac4`), bringing workspace runtime improvements, heartbeat tooling hardening, issue detail UX improvements, and new sidebar preference persistence.
- **Test environment fix** — Fixed `localStorage.clear is not a function` test failures caused by Node.js 25's built-in localStorage conflicting with jsdom tests.
