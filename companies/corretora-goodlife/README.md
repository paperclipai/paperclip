# Corretora Goodlife — Agent Company

Esta package contém a definição da organização de agentes que trabalharão no
projeto do site da Corretora Goodlife.

- Company slug: `corretora-goodlife`
- Projeto principal: `site-goodlife` (local: `D:/TestesIA/site-goodlife`, repo: https://github.com/corretora-goodlife/site-goodlife.git)

Getting started

1. Importe o pacote para a sua instância Paperclip (UI -> Companies -> Import) ou use:

```
npx paperclipai company import --from companies/corretora-goodlife
```

2. Configure secrets necessários (por exemplo `GH_TOKEN`) no painel da company.
3. Ajuste os `Working directory` de cada agente para `D:/TestesIA/site-goodlife`.

Agentes incluídos: `cto`, `dev-fullstack`, `pentester`, `designer-ux-ui`, `qa`, `devops`.
