## ucpm-default — canonical UCPM property-management company

This is the canonical default UCPM property-management company. Every property loaded into the UCPM-Paperclip system inherits this SOP and agent set by default; per-property companies (e.g. `companies/123-main-st/`) may override individual procedures or values but inherit everything they do not override. This template is not itself a runnable instance — see `state.yml`.
