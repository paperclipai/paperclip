# ip plugin

This directory is the install target for Anthropic's open-source legal practice-area
plugin for **ip** (announced 2026-05-12, Apache 2.0).

The corresponding Practice Lead is at `agents/practice-leads/ip-lead.md`. The
Practice Lead treats this plugin as its toolbelt.

## Install

When Anthropic's plugin lands publicly, install via:

```bash
pnpm odysseus plugins install anthropic/legal-ip
```

Until then, this directory is a documented placeholder.
