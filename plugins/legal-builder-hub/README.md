# legal-builder-hub plugin

This directory is the install target for Anthropic's open-source legal practice-area
plugin for **legal-builder-hub** (announced 2026-05-12, Apache 2.0).

The corresponding Practice Lead is at `agents/practice-leads/legal-builder-hub-lead.md`. The
Practice Lead treats this plugin as its toolbelt.

## Install

When Anthropic's plugin lands publicly, install via:

```bash
pnpm odysseus plugins install anthropic/legal-legal-builder-hub
```

Until then, this directory is a documented placeholder.
