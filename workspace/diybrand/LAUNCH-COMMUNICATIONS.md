## Email Template Testing Checklist

Before sending any launch communications, verify rendering in:

```
Desktop Email Clients:
☐ Gmail (web)
☐ Outlook (web)
☐ Apple Mail (desktop)

Mobile Email Clients:
☐ Gmail (mobile)
☐ Apple Mail (iOS)
☐ Outlook (mobile)

Email Service Provider:
☐ Email previews in Brevo/Mailchimp/Klaviyo
☐ Links clickable and working
☐ Images loading correctly
☐ Mobile responsiveness
☐ Spam filter testing (sendtest to SpamAssassin)
```

**Common Issues to Check:**
- Images breaking on mobile
- Links not clickable
- Text too small on mobile
- CTA buttons responsive
- Footer links working
- Unsubscribe link present
