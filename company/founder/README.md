# Founder

이 폴더는 대표 개인의 `철학`, `문제의식`, `서사`, `장기 동기`를 보관하는 영역이다.

핵심은 하나다.

`회사의 장기 방향은 대표 개인의 의미와 문제의식에서 시작되므로, 이 맥락을 분리해서 보관한다.`

## 이 폴더에 들어갈 것

- 대표 개인의 의미와 철학 정리
- 창업 동기 초안
- 장기 비전의 개인적 배경
- 회사 문서로 바로 승격되기 전의 메모

## 운영 원칙

- 회사 공식 기준과 대표 개인 메모를 섞지 않는다
- 이 폴더의 문서는 Strategy의 입력값이 될 수는 있지만, 자동으로 공식 전략이 되지는 않는다
- 공식 전략으로 승격되면 해당 내용을 `strategy/` 또는 `management/`에 다시 정리한다
