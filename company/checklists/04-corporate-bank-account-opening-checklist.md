## 1. 공통 필수 서류

아래는 은행이 거의 공통으로 보는 기본 세트다.

- [ ] 사업자등록증 원본 또는 사업자등록증명원
- [ ] 법인등기사항전부증명서
  - 말소사항 포함
  - 발급 3개월 이내로 준비
- [ ] 법인인감증명서
  - 발급 3개월 이내로 준비
- [ ] 법인인감도장
- [ ] 대표자 신분증
  - 주민등록증 또는 운전면허증
- [ ] 계좌 개설 목적 설명
  - 운영자금
  - 매출 입금
  - 세금 납부
  - 급여 / 외주비 이체

## 2. 은행이 자주 추가로 요구하는 서류

은행과 법인 상태에 따라 아래가 추가된다.

- [ ] 주주명부 또는 주식등변동상황명세서
  - 지분현황 확인용
- [ ] 정관
- [ ] 실소유자 확인서류
  - 대표자 선출 회의록
  - 주주구성 확인서류
  - 대표권 확인 가능한 문서
- [ ] 사용인감계
  - 법인인감 말고 사용인감으로 처리할 경우
- [ ] 위임장
  - 대표자 본인 방문이 아니면 필요
  - 보통 법인인감 날인 요구
- [ ] 대리인 신분증
- [ ] 공동대표 관련 추가서류
  - 공동대표 체계면 지점마다 요구가 조금 다름
  - 방문 전 지점 확인 필요

## 3. 금융거래 목적 증빙

요즘은 `정상 영업 목적`이 확인되지 않으면
계좌 개설이 거절되거나 `금융거래한도계좌`로 열릴 수 있다.

아래에서 가능한 것을 최대한 많이 챙기는 편이 안전하다.

- [ ] 사무실 임대차계약서
- [ ] 홈페이지 또는 서비스 소개 페이지
- [ ] 진행 중인 계약서 또는 견적서
- [ ] 세금계산서 / 인보이스 / 발주서
- [ ] 사무실 집기 / 인테리어 구입 영수증
- [ ] 간판 사진 또는 포털 로드뷰 확인 가능 자료
- [ ] 창업지원기관 발급 확인서류
- [ ] 은행 실사 대응 가능 상태
  - 사업장 주소
  - 간판 / 우편물 / 출입 가능 여부

## 4. 비브라이트코드에서 먼저 꺼낼 위치

repo 기준으로 원본은 Drive에 있어야 한다.
아래 순서로 찾는 것이 맞다.

### 기본 법인서류

- [ ] Google Drive `비브라이트코드 > 법인자료`
  - 사업자등록증
  - 법인등기사항전부증명서
  - 법인인감증명서
  - 정관
  - 주주명부
  - 사용인감계

### 목적증빙

- [ ] Google Drive `비브라이트코드 > 계약서`
  - 수주 계약서
  - 외주 계약서
  - 업무 계약서
- [ ] Google Drive `비브라이트코드 > 세금`
  - 세금계산서
  - 납세 관련 자료

## 5. 방문용 실무 패키지

실제로는 아래처럼 한 번에 챙기면 편하다.

- [ ] 원본 1세트
- [ ] 사본 2세트
- [ ] 휴대폰 PDF 백업본
- [ ] 대표자 휴대폰
- [ ] 공동인증서 또는 인증수단
- [ ] 법인카드 / 인터넷뱅킹 / OTP 동시 신청 여부 메모
- [ ] 첫 입금 예정 금액 메모

## 6. 은행 가기 전 마지막 확인

- [ ] 어느 은행 / 어느 지점으로 갈지 정한다
- [ ] 대표자 본인 방문인지, 대리인 방문인지 정한다
- [ ] 단독대표인지, 공동대표인지 확인한다
- [ ] 개설 목적을 한 문장으로 정리한다
  - 예: `법인 운영비 집행과 고객 대금 입금용 주거래 계좌 개설`
- [ ] 인터넷뱅킹까지 같이 열지 정한다
- [ ] 법인카드까지 같이 신청할지 정한다
- [ ] 해당 지점에 전화해서 아래 3가지를 재확인한다
  - 오늘 기준 추가 필요서류
  - 공동대표 / 대리인 처리 가능 여부
  - 목적증빙으로 인정되는 자료

## 한 줄 규칙

`등기 + 인감 + 대표자 신분증 + 목적증빙` 4종 세트를 먼저 잡고 가면 실패 확률이 크게 줄어든다.
