# Management

이 폴더는 회사의 `경영지원`, `HR`, `수주 판단`, `대외 운영`을 정리하는 영역이다.

## 이 폴더에 들어갈 것

- 외주 모집용 메인 사이트 문구
- 회사 소개와 대외 메시지 기준
- 채용공고와 채용 철학
- HR 운영 기준
- 채용/배치 기준
- 수주 판단 기준
- 외주/계약/견적 운영 기준
- 경영지원 체크 문서

## 운영 원칙

- 대외 메시지도 전략 문서와 연결해서 관리한다
- 채용공고도 운영 기준 문서로 본다
- 사람 문제를 채용으로 먼저 풀지 않는다
- 수주는 매출보다 구조 적합성을 본다
- 경영지원도 원칙과 템플릿으로 운영한다
