# Decisions

이 폴더는 중요한 `의사결정 기록`을 남기는 영역이다.

## 이 폴더에 들어갈 것

- 전략적 방향 전환
- 도구 선택
- 팀 운영 방식 변경
- 제품 우선순위 변경
- 큰 프로젝트 수주/거절 판단

## 기본 템플릿

- 결정 일자
- 결정 내용
- 배경
- 다른 대안
- 왜 이 결정을 했는가
- 이후 점검 시점

## 운영 원칙

- 중요한 결정은 기억에 맡기지 않는다
- 이유까지 남겨야 나중에 흔들리지 않는다
