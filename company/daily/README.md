# Daily

이 폴더는 `매일 운영 메모`와 `짧은 회고`를 남기는 영역이다.

## 이 폴더에 들어갈 것

- 대표 일일 메모
- Daily Focus 문서
- 오늘의 우선순위
- 막힌 점
- 짧은 회고
- 내일의 단일 초점

## 운영 원칙

- 길게 쓰지 않는다
- 오늘의 판단과 내일의 초점만 남긴다
- 일간 기록은 주간 점검의 재료가 된다
