- Weekly Cycle 템플릿
- Daily Focus 템플릿
- 회고 템플릿
- 1:1 템플릿
- 수주 판단 템플릿
- 제안서 템플릿
- 일간/주간/월간 점검 템플릿
- 자금 보드 템플릿
- AI 사용량 보고 템플릿

## 운영 원칙

- 반복되는 것은 템플릿으로 만든다
- 템플릿은 길게 설명하지 말고 바로 쓸 수 있어야 한다
- 고객 발송 템플릿은 `짧은 결론 + 쉬운 상세 설명` 구조를 유지한다
