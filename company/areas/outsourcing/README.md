# Outsourcing Area

이 폴더는 회사의 `1단계 외주개발` 영역 문서를 모으는 곳이다.

## 이 영역의 역할

- 현금 흐름 확보
- 고객 접점 확보
- 반복 가능한 문제 발견
- 제품 자산으로 이어질 수 있는 일만 선별해서 받기

## 이 영역에 들어갈 것

- 리드 대응 워크플로우
- 수주 판단 운영 기준
- 견적 / 제안 운영 기준
- 외주개발 고객 대응 방식
