# Flotter

이 폴더는 `자체서비스 개발 영역 안에서 Flotter를 어떻게 볼지` 정리하는 입구다.

핵심은 하나다.

`Flotter는 2단계 자체서비스 개발의 대표 제품 축이다.`

## Flotter를 여기 두는 이유

- Flotter는 외주개발이 아니라 자체서비스 개발에 해당한다
- 회사의 3단계 실행축 기준으로 보면 `2단계 자체서비스 개발`의 핵심 제품이다
- 따라서 실행 관점에서는 `own-services` 아래에서 먼저 보여야 한다
