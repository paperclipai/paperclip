# Areas

이 폴더는 회사의 `3단계 실행축` 기준으로 문서를 나누는 영역이다.

핵심은 하나다.

`기능별 폴더와 별도로, 회사가 실제로 키우려는 사업 영역 기준으로 문서를 묶는다.`

## 3단계 실행축

1. 외주개발
2. 자체서비스 개발
3. 게임개발사

## 각 영역의 역할

### 1. 외주개발

- 지금 당장 현금 흐름을 만드는 영역
- 고객 접점을 만들고 문제를 발견하는 영역
- 리드 대응, 수주 판단, 견적, 제안 흐름을 다루는 영역

### 2. 자체서비스 개발

- 반복 매출과 자산을 만드는 영역
- 제품 구조, 고객 workflow, 운영 구조를 쌓는 영역

### 3. 게임개발사

- 장기적으로 가장 큰 상방을 만드는 영역
- IP, 게임 제작 역량, 실험작과 장기 자산을 다루는 영역
