# Products

이 폴더는 회사의 `제품별 원본 문서`, `사업계획서 아카이브`, `제품 컨텍스트`를 보관하는 영역이다.

핵심은 하나다.

`회사 전체 전략과 제품별 원본 문서를 섞지 않고, 제품 단위 맥락을 따로 보관한다.`

## 이 폴더에 들어갈 것

- 제품별 사업계획서 아카이브
- 제품별 노션 원본 링크와 source of truth
- 제품별 포지셔닝 초안
- 제품별 현재판 실행 문서
- 제품별 과거 가설과 현재 해석 차이

## 운영 원칙

- 제품 원본은 원문 맥락을 최대한 유지한다
- 현재 회사 전략과 충돌하는 부분은 별도 해석으로 표시한다
- 노션 원본 링크와 repo 아카이브를 함께 남긴다
