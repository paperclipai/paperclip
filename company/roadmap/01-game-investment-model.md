# Game Investment Model

기준일: 2026-03-16

## 목적

이 문서는 `남는 시간에 게임도 해보자` 수준이 아니라  
회사의 장기 방향을 위한 `전략 투자 시간`을 어떻게 운영할지 정리한 문서다.

핵심은 두 가지다.

1. 게임 방향을 감정이나 피로도에 맡기지 않는다
2. 투자 시간이 끝나도 자산이 남도록 설계한다

## 기본 원칙

게임 관련 시간은 `남는 시간`이 아니라 `고정 투자 시간`으로 본다.

이렇게 해야 하는 이유:

- 남는 시간으로 두면 항상 밀린다
- 실제로는 아무것도 쌓이지 않는다
- 팀의 장기 방향성이 말로만 남는다
- 박상훈 같은 인력의 장기 동기와도 연결이 약해진다

따라서 게임 관련 시간은  
`시간이 남으면 하는 일`이 아니라  
`작게라도 계속 투자하는 일`로 운영한다.

## 2026년 투자 방향

2026년의 게임 투자 시간은 아래 비율로 운영한다.

- 게임 제작 도구: 70%
- 소형 게임 프로토타입: 30%

이 비율을 추천하는 이유:

- 도구는 superbuilder, 스토리 SaaS, 향후 게임 제작까지 모두 자산으로 남는다
- 프로토타입은 팀의 감각, 방향성, 실제 제작 감각을 유지시킨다
- 도구만 하면 내부 개발 병이 오고
- 프로토타입만 하면 재사용 자산이 약해진다

## 투자 시간 운영 방식

### 주간 기준

- 주당 6~8시간을 `전략 투자 시간`으로 고정한다
- 가능하면 주 1회 반나절 또는 2회 블록으로 캘린더에 먼저 잡는다
- "이번 주 바쁘니까 생략"은 예외로만 허용한다

### 월간 기준

- 매 4주마다 주제는 1개만 잡는다
- 한 달 동안 도구/프로토타입 테마를 섞더라도 중심 주제는 하나만 둔다
- 주제가 끝나면 다음 달에 바꾼다

예시 주제:

- 스토리/퀘스트 데이터 구조
- 대사/이벤트 작성 툴
- 로컬라이징 반영 파이프라인
- 캐릭터/세계관 일관성 관리
- 게임용 콘텐츠 작성 에디터
- 작은 전투 루프 프로토타입

## 투자 결과물 기준

게임 투자 시간은 반드시 아래 결과물 중 하나로 끝나야 한다.

- 실제로 실행되는 도구
- 내부에서 바로 쓰는 워크플로우
- 플레이 가능한 짧은 프로토타입
- 다음 달에도 재사용 가능한 데이터 구조
- 문서화된 제작 규칙

즉, `생각만 정리한 시간`으로 끝나면 안 된다.

## 무엇에 투자할 것인가

### 우선순위 1. 게임 제작 도구

가장 먼저 투자할 것은 `게임 제작에 바로 쓰이는 도구`다.

권장 주제:

- 스토리/퀘스트/대사 데이터 구조
- 세계관/캐릭터 관리 툴
- 이벤트/분기 설계 도구
- 로컬라이징/검수 파이프라인
- 게임 콘텐츠 작성용 에디터
- 내부 dogfooding 가능한 제작 워크플로우

### 우선순위 2. 소형 게임 프로토타입

도구만 만들면 실제 제작 감각이 죽기 때문에  
작은 프로토타입도 병행한다.

원칙:

- 2~4주 안에 끝나는 범위
- 한 가지 핵심 재미만 검증
- 내부 도구를 바로 써볼 수 있어야 함

예시:

- 대사/이벤트 중심 짧은 플레이 루프
- 선택지와 상태 변화를 검증하는 프로토타입
- 캐릭터/서사 전달감을 보는 짧은 시퀀스

## 팀 배치 관점

### 대표

- 구조 정의
- 제작 워크플로우 설계
- 도구 방향성 판단
- 작은 프로토타입의 핵심 설계

### 박지수

- 에디터/툴 UI
- 플레이어가 직접 만지는 화면
- 프로토타입의 시각적 전달력 정리

### 박상훈

- 시스템성 있는 게임 로직
- 규칙, 상태, 시뮬레이션
- 게임성과 연결되는 구조 문제

핵심은 전원이 게임을 "각자 따로" 만드는 게 아니라  
Builder 팀으로 같이 쌓되, 상대적 강점이 다른 영역을 맡는 것이다.

## 판단 기준

아래 4개 중 3개 이상 만족하면 좋은 투자다.

1. 향후 자체 게임 제작에 직접 쓰일 수 있다
2. 스토리 SaaS나 superbuilder에도 응용 가능하다
3. 한 달 안에 의미 있는 결과물로 남길 수 있다
4. 특정 개인의 취미 프로젝트로 끝나지 않는다

## 하지 말아야 할 것

### 하지 말 것 1

`남는 시간` 개념으로 미루기

### 하지 말 것 2

만능 게임 엔진, 만능 퀘스트 시스템 같은 범용 툴 욕심

### 하지 말 것 3

너무 큰 게임을 먼저 만들려는 시도

### 하지 말 것 4

도구도 없고 자산도 안 남는 감정형 프로토타이핑 반복

### 하지 말 것 5

외주와 제품 일정이 흔들리는데도 게임 시간을 감정적으로 늘리기

## 2026년 권장 운영 문장

`게임은 남는 시간에 하는 취미가 아니라, 도구와 프로토타입으로 자산이 남는 전략 투자 시간으로 운영한다.`

## 대표용 한 줄 규칙

`게임 시간은 언젠가의 꿈이 아니라, 매주 쌓이는 자산으로 보여야 한다.`
