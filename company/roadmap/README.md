# Roadmap

이 폴더는 회사의 `장기 로드맵`, `연간 계획`, `챕터 계획`을 정리하는 영역이다.

## 이 폴더에 들어갈 것

- 3년/5년 장기 로드맵
- 연간 운영 계획
- Chapter 계획과 회고
- 제품/게임/수익 구조 로드맵

## 운영 원칙

- 로드맵은 희망회로가 아니라 상태 변화 중심으로 쓴다
- 한 챕터의 목표는 2~3개를 넘기지 않는다
- 장기 방향과 단기 실행이 연결되어야 한다
