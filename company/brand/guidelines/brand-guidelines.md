# Brand Guidelines

## 슬로건

> 전세계의 게이머들에게 서사깊은 스토리를 언어의 장벽없이 경험하게 하자!

이 슬로건이 BBrightCode의 존재 이유이자, Flotter가 탄생한 이유다.

## 1. 로고

### 구조

로고는 다음 요소들의 조합으로 구성된다:

```
[레이어 3 - 외곽 다이아몬드]
  [레이어 2 - 중간 다이아몬드]
    [레이어 1 - 내부 다이아몬드 + 사각형]
      [파이프라인 3개]
```

### 형태와 의미

**다이아몬드 — 구조적이지만, 규칙을 비튼다**

사각형을 쌓아 90도 회전시켜 만든 다이아몬드 형태. 구조적이고 규격화되어 있지만, 기존의 룰을 비튼다.

**겹겹이 쌓인 레이어 — 빌더(Builder)**

여러 겹으로 쌓은 구조적인 형태는 빌더를 의미한다.

**별 — 빛나는 형태**

가운데 블록까지 포함한 전체 형태는 별이 빛나는 모습을 표현한다.

**3개의 파이프라인 — 미션에 도달하는 3단계**

가운데 3개의 긴 블록은 3단계에 걸쳐 미션에 도달하는 여정을 표현한다:

| 파이프라인 | 색상 | 의미 |
|-----------|------|------|
| 1단계 (가장 희미한) | Gray 400 (#9CA3AF) | 현재의 우리 |
| 2단계 (중간) | Gray 300 (#D1D5DB) | 우리가 만드는 제품 |
| 3단계 (가장 밝은) | Gray 50 (#F9FAFB) | 최종 목표 — 가장 고결한 방향 |

**축소 시 — 톱니/기어**

로고를 작게 축소하면 톱니바퀴, 기어처럼 보인다. 우리의 깊은 DNA는 결국 엔지니어이기 때문이다.

### 로고 색상 구성

로고에 사용된 색상은 Tailwind CSS Gray 계열을 기반으로 한다:

| 색상 | HEX | 용도 |
|------|-----|------|
| Gray 900 | `#111827` | 내부 다이아몬드, 내부 사각형 (핵심 코어) |
| Gray 700 | `#374151` | 외곽 다이아몬드, 외곽 사각형 (기반 레이어) |
| Gray 50 | `#F9FAFB` | 중간 다이아몬드, 중간 사각형, 메인 파이프라인 (밝은 레이어, 목표) |
| Gray 300 | `#D1D5DB` | 두 번째 파이프라인 (제품) |
| Gray 400 | `#9CA3AF` | 세 번째 파이프라인 (현재) |

### 로고 버전

| 파일 | 설명 |
|------|------|
| `logo-symbol.svg` | 기본 심볼 (컬러) |

### 사용 규칙

- **최소 여백**: 로고 심볼 높이의 1x를 사방에 확보
- **최소 크기**: 24px (디지털), 10mm (인쇄)
- **배경**: 밝은 배경 또는 어두운 배경 모두 사용 가능 (단, 대비 확보 필수)

### 금지 사항

- 로고를 임의로 회전하지 않는다
- 로고 색상을 임의로 변경하지 않는다
- 로고를 비율 무시하고 늘리거나 줄이지 않는다
- 로고 위에 텍스트나 다른 요소를 겹치지 않는다
- 그림자, 그라데이션, 외곽선 등 효과를 추가하지 않는다

## 2. 색상 시스템

### Primary Palette

로고에서 파생된 Gray 톤 기반 팔레트:

```
이름:    Core Black
HEX:    #111827
RGB:    17, 24, 39
용도:    핵심 브랜드 컬러, 주요 텍스트, 강조 배경

이름:    Foundation Gray
HEX:    #374151
RGB:    55, 65, 81
용도:    보조 배경, 서브 텍스트

이름:    Light Surface
HEX:    #F9FAFB
RGB:    249, 250, 251
용도:    밝은 배경, 카드, 표면
```

### Extended Palette

```
이름:    Pipeline Current (Gray 400)
HEX:    #9CA3AF
RGB:    156, 163, 175
용도:    비활성 상태, 보조 텍스트, 현재 상태 표현

이름:    Pipeline Product (Gray 300)
HEX:    #D1D5DB
RGB:    209, 213, 219
용도:    구분선, 보더, 중간 상태 표현

이름:    Pipeline Goal (Gray 50)
HEX:    #F9FAFB
RGB:    249, 250, 251
용도:    목표 상태, 최고 강조, 하이라이트
```

### 색상 비율

- **60%** — Light Surface (#F9FAFB): 배경과 여백
- **30%** — Core Black (#111827): 텍스트와 주요 요소
- **10%** — Foundation Gray (#374151): 액센트와 보조 요소

## 3. 디자인 철학

> "구조적이지만, 기존의 규칙을 비튼다."

- **빌더**: 겹겹이 쌓아 올리는 구조 — 우리는 만드는 사람들이다
- **모노크롬**: 화려함 대신 깊이와 단계로 표현
- **방향성**: 현재 → 제품 → 목표, 3단계에 걸쳐 점진적으로 밝아지는 미션 여정
- **엔지니어 DNA**: 축소했을 때 기어/톱니로 보이는 형태 — 본질은 엔지니어링
