Run the monthly compliance posture check across ISO 27001, SOC 2 Type II, and GDPR. Update the evidence register and deliver the compliance dashboard to ShieldCEO.

## Checklist

1. Review all ISO 27001 controls: flag any that have lapsed or require evidence refresh
2. Review SOC 2 TSC controls: verify continuous evidence is still being collected for each criterion
3. GDPR: check Article 30 register for any new processing activities requiring documentation
4. Identify any new control gaps introduced by system changes or incidents in the past month
5. Update the compliance posture dashboard (% controls passing by framework)
6. Draft monthly compliance report: posture score, new gaps, remediation progress
7. Post report as task comment and @-mention ShieldCEO for review
8. Escalate any control failure that creates immediate audit risk to ShieldCEO
