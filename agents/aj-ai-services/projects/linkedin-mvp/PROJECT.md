The LinkedIn MVP project delivers the first working version of the AI-powered LinkedIn post generator. The goal is a fully functional product — from prompt input to formatted post output — shipped within 24 hours of kickoff.

## Deliverables

1. LinkedIn post generator API (core module)
2. GitHub Actions CI/CD pipeline
3. Content calendar and posting strategy
4. Brand templates for LinkedIn posts
5. AI ethics guardrails for generated content
6. Security baseline for infrastructure
7. Project tracking and delivery cadence

## Success Criteria

- Users can input a topic and receive a formatted LinkedIn post with hashtags
- CI pipeline runs on every commit
- At least 3 content templates ready for use
- All generated content passes the Responsible AI checklist
