Run the daily standup for AJ AI Services Pvt Ltd.

## Agenda

1. **Yesterday** — What did each team complete?
2. **Today** — What is each team working on?
3. **Blockers** — Any impediments to progress?
4. **MVP Status** — Are we on track for the 24-hour target?

## Steps

1. Pull the latest status of all open issues from Paperclip
2. Check progress against the MVP milestone targets
3. Identify any overdue tasks or blockers
4. Post a standup summary comment to the daily standup issue (or create one if none exists)
5. Escalate blockers to the appropriate lead

## Output

A standup summary posted in Paperclip with:
- Red/Yellow/Green status for each team
- Blockers identified with owner and resolution plan
- Updated ETA for MVP delivery
