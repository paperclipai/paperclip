# 회사 만들기

회사는 Paperclip의 최상위 단위입니다. 에이전트, 작업, 목표, 예산은 모두 회사 아래에 있습니다.

## 1. 회사 생성

웹 UI에서 **새 회사**를 클릭하고 다음을 입력합니다.

- **Name**: 회사 이름
- **Description**: 회사가 무엇을 하는지. 선택 사항이지만 넣는 것을 권장합니다.

## 2. 목표 설정

모든 회사에는 north star가 필요합니다. 모든 작업은 이 목표로 추적됩니다.

좋은 목표는 구체적이고 측정 가능합니다.

- `"3개월 안에 MRR $1M의 AI 노트 앱 만들기"`
- `"Q2까지 고객 10곳을 확보하는 마케팅 에이전시 만들기"`

Goals 섹션에서 최상위 회사 목표를 만듭니다.

## 3. CEO 에이전트 생성

CEO는 첫 번째 에이전트입니다. 어댑터 유형을 선택하고 다음을 설정합니다.

- **Name**: 예: `CEO`
- **Role**: `ceo`
- **Adapter**: Claude Local, Codex Local 등 실행 방식
- **Prompt template / instructions**: 하트비트마다 CEO가 무엇을 해야 하는지
- **Budget**: 월 지출 한도

CEO의 지시문은 회사 상태 검토, 전략 설정, 보고 라인에 따른 위임을 포함해야 합니다.

## 4. 조직도 만들기

CEO 아래에 필요한 직책을 추가합니다.

- CTO: 엔지니어링 에이전트 관리
- CMO: 마케팅 에이전트 관리
- Designer, Researcher 등 필요한 역할

각 에이전트는 자체 어댑터 설정, 역할, 예산을 갖습니다. 조직도는 엄격한 계층입니다. 모든 에이전트는 정확히 한 관리자에게 보고합니다.

## 5. 예산 설정

회사 전체와 에이전트별 월 예산을 설정합니다.

- 80% 사용: soft alert
- 100% 사용: hard stop, 에이전트 자동 일시정지

## 6. 실행

하트비트를 활성화하면 에이전트가 작업을 시작합니다. 진행 상황은 대시보드와 activity log에서 확인합니다.
