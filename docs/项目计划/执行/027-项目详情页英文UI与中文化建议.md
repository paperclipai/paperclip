# 项目详情页：右侧内容区英文 UI 与中文化建议

## 范围

- **场景**：左侧边栏「项目」下选中某一项目后，**右侧主内容区**（项目标题区、Tab、各 Tab 内嵌组件、以及同一流程内弹出的对话框）。
- **排除**：
  - **预算** Tab 内 `BudgetPolicyCard`：文案已走 `budgetPolicyUi`（中文为主）。
  - 用户/数据字段：项目名称、issue 标题、分支名、URL、插件 `displayName`、后端返回的 `action.label` / `action.description` / `blockingReasons` / `warnings` 等（若后端为英文则需另做 API 或映射层）。
  - 面包屑第一格 **Projects** 若仅出现在全局顶栏：见文末「附录」；实现时可与详情区一并收进 i18n。
