### 任务 3：变基并处理可能的冲突

**文件：**
- 操作：变基到远程 master

- [ ] **步骤 1：执行变基操作**

运行：
```bash
git rebase origin/master
```

- [ ] **步骤 2：如果出现冲突，解决冲突**

如果出现冲突，执行以下步骤：

1. **查看冲突文件**
```bash
git status
```

2. **手动解决冲突**，编辑冲突文件，保留需要的代码

3. **标记冲突已解决**
```bash
git add <解决冲突的文件>
```

4. **继续变基**
```bash
git rebase --continue
```

重复步骤 1-4 直到所有冲突解决

- [ ] **步骤 3：验证变基成功**

运行：
```bash
git log --oneline -5
```

预期输出：最新的几个提交应该是本地提交，之前的提交来自 origin/master
