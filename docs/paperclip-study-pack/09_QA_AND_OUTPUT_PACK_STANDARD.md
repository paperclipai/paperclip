# QA and Output Pack Standard

## Purpose

Define the final review and packaging standard for SINK DINK upload-ready media packs.

## Final output package

```
final_upload_pack/
├── reels/
├── carousels/
├── captions/
├── preview/
├── qa/
└── upload_checklist.md
```

## QA checklist

### Content

- clear hook
- clear message
- correct tone
- audience fit
- no unnecessary filler

### Visuals

- readable text
- mobile friendly
- clear contrast
- consistent layout
- correct file size and format

### Reels

- strong first three seconds
- clean subtitle timing
- clear audio plan
- correct aspect ratio
- final MP4 export

### Carousels

- strong first slide
- clear slide sequence
- readable text
- final PNG or JPG export

### Captions

- clear caption
- relevant hashtags
- simple CTA
- platform-ready formatting

## Approval labels

Use these labels in reports:

- Draft
- Review Needed
- Approved for Manual Upload
- Needs Fix

## Upload checklist

Before manual upload, confirm:

- all files exist
- filenames are clear
- captions are ready
- hashtags are ready
- preview reviewed
- QA report included

## Completion rule

A pack is complete only when all media files and supporting text files are present in the final folder.
