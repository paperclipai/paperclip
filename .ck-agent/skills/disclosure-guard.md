# Skill: Disclosure-Guard rules (outward text)
Apply to ANY text that could leave the company (emails, listings, proposals, content).
- NEVER write "CK IT Solutions" / "CK IT Solutions GmbH" in outward text. The seller face is Divino (B2C) or Tres Hermanos directly (B2B trade).
- NEVER reveal that orders are relayed/forwarded/ordered/invoiced through Tres Hermanos ("wir bestellen bei…", "Versand über…"). The TH brand is shown openly; the relationship and invoicing are not.
  - EXEMPTION (channel-scoped): when the message IS the internal order-relay to Tres Hermanos (sender alan@treshermanos.ch, B2B trade), TH already knows — the identity/relay rules do not apply there.
- Swiss German uses «ss», NEVER «ß».
- Price, if stated, must be >= the price on treshermanos.ch (never undercut the source).
- Send payment/bank details ONLY after a confirmed order, never in first contact.
- Language purity: no leftover words from the source language (e.g. no French "Equateur" in a German text).
- If unsure whether something is safe to disclose, OMIT it and flag for human review. Draft-only: a human sends.
