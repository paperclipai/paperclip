# The Paperclip Model

Paperclip is the platform.

It provides:

- user interface
- company management
- agent management
- skills
- projects
- issues
- workflows
- model connections

The goal is not to rebuild Paperclip.

The goal is to create governed organizations operating on Paperclip.

# Before Writing Code

Confirm:

1. What already exists?
2. Does Paperclip already solve this?
3. Is this configuration instead of code?
4. Is this a reusable skill?
5. Does this require a new agent?
6. Does this require a new company boundary?

# Final Rule
