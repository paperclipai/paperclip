- [ ] Disperser (30–50 HP) — RPM calibration verified, target ~1400 RPM
- [ ] Impeller installed: 300 mm diameter
- [ ] Grind vessel (500 L SS) — clean, dry, drain valve closed
- [ ] Letdown vessel (500 L SS) — clean, dry, anchor agitator OK
- [ ] Transfer pump (diaphragm/progressive cavity) — primed and ready
- [ ] In-line 100 µm bag filter — installed, housing sealed
- [ ] Filling machine — semi-automatic, clean, purge tested
- [ ] Temperature probe — calibrated in each vessel
- [ ] Tachometer — reading confirmed

- [ ] All raw materials weighed and staged (within ±0.5% of target)
- [ ] RM QC release status verified for all lots (PAL-21-QC-001)
